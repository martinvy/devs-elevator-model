tracesUser(time, aDEVS, variable, value)[source]¶

Perform all tracing actions for a user imposed modification. This is NOT supported by default DEVS, so we don’t require tracers to handle this either.

Parameters:
  • time – the time at which the modification happend; this will be the termination time of the previous simulation run and not the time at which the timeAdvance was recomputed!
  • aDEVS – the atomic DEVS model that was altered
  • variable – the variable that was altered (as a string)
  • value – the new value of the variable
